Basics

  • let allows you to label a variable that can be changed throughout the course of the script, and probably allows input from the user (assuming I will learn about this in future lessons)
  • constlabels a constant
  • alert(..) produces an announcement box displaying a message

Example:

1 let message = "Hello";
2 alert(message);

  • As expected, coding with numbers works as expected. The only exception is when using '+'. The addition function only works when combing numbers with numbers or strings with strings. If you try to combine, you will get a string result. This is because '+' is used for both number operations and string operations (concatenate).
  • All other operations will try to convert strings to numbers to complete them.

Open the developer tools and use the console to explore the following script (already embedded in the page).

let b = 7 * a;
const max = 57;
let actual = max - 13;
let percentage = actual / max;
let c = percentage;

Types of Data

  1. Numbers - pretty self-explanatory; they are "safe" in JS
    • Infinity & -Infinity
    • NaN (not a number) - NaN in any operation will always return NaN (except NaN ** 0 ("not a number" to the 0 power = 1)
  2. BigInt (big integers) - used to represent numbers larger than ±2^53-1 (or 9007199254740991); it's created by appending n to an integer
  3. Strings - items surrounded by quotes
    • Double & single quotes are used for exact statements
    • Backtick quotes (`Hello`)are extended functionality quotes and can be used to embed variables and expression into a string by wrapping the in ${}. The formula is calculated and the result becomes part of the string. This only happens in backticks.
  4. Boolean - used for storing true/false results, like if data points were checked or comparing two numbers
  5. The "null" value - used to represent unknown values
  6. The "undefined" value - used to represent values not assigned
  7. Objects - used to store collections of data and more complex entities
  8. Symbols - used with Objects to create unique identifiers for them
  9. The typeof operator - returns the type of operator in a statement
