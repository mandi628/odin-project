Javascript Notes

Basics

  • let allows you to label a variable that can be changed throughout the course of the script, and probably allows input from the user (assuming I will learn about this in future lessons)
  • constlabels a constant
  • alert(..) produces an announcement box displaying a message

Example:

1 let message = "Hello";
2 alert(message);

  • As expected, coding with numbers works as expected. The only exception is when using '+'. The addition function only works when combing numbers with numbers or strings with strings. If you try to combine, you will get a string result. This is because '+' is used for both number operations and string operations (concatenate).
  • All other operations will try to convert strings to numbers to complete them.

Open the developer tools and use the console to explore the following script (already embedded in the page).

let a = 10;
let b = 7 * a;
const max = 57;
let actual = max - 13;
let percentage = actual / max;
let c = percentage;
